# Учёт обращений студентов

## Описание проекта
Данное веб-приложение предназначено для демонстрации базовых возможностей JavaScript в рамках дисциплины  
**«Разработка web-приложений на языке JavaScript»**.

Приложение реализует простую систему учёта обращений студентов:
- просмотр списка обращений  
- фильтрация обращений  
- вывод статистики  

---

## Функциональность

- Показ всех обращений  
- Фильтрация по статусу (новые)  
- Подсчёт статистики  
- Работа с массивами данных  
- Использование условий и циклов  

---

## Используемые технологии

- HTML5  
- CSS3 (минимально)  
- JavaScript (ES6+)

## Основные методы JavaScript

В проекте используются:

- Переменные (`let`, `const`)  
- Массивы и объекты  
- Приведение типов (`Number`)  
- Условные конструкции (`if`, `switch`)  
- Циклы (`for`)  
- Методы массивов (`filter`)  
- Работа с DOM
