## 2. Описание предметной области

Система предназначена для регистрации обращений студентов в образовательной организации. Она позволяет фиксировать обращения, отслеживать их статус и анализировать типы проблем.  

Решаемые задачи:
- учёт обращений студентов  
- контроль статусов обработки  
- упрощение взаимодействия студентов и администрации  
- анализ частых проблем  

---

## 3. Пользователи системы

| Роль пользователя | Описание функций                                        |
|-------------------|---------------------------------------------------------|
| Студент           | Отправляет обращение (в перспективе через форму).       |
| Оператор          | Регистрирует обращения и обновляет их статусы.          |
| Администратор     | Управляет системой и справочниками.                     |

---

## 4. Основные сущности предметной области (данные)

| Сущность     | Описание                                      | Основные атрибуты                                  |
|-------------|-----------------------------------------------|----------------------------------------------------|
| Студент     | Пользователь, создающий обращение             | ФИО, группа, контактные данные                    |
| Обращение   | Запрос или проблема студента                  | ID, дата, тема, описание, статус, приоритет       |
| Сотрудник   | Обрабатывает обращения                        | ФИО, должность, список обращений                  |
